Mac Cleaner

Mac Cleaner is free software for cleaning cache and system optimization.

How to install

1. Download Mac Cleaner from home pages.

2. Find the DMG file in your Downloads folder, and open it.

3. Drag Mac Cleaner and drop it to the Applications folder.

Run it

Open from Launchpad and click Mac Cleaner.

If you are first time use Mac Cleaner Mac will prompt the following picture. Don't worry, please follow the steps.

1. Open System Preferences

2. Click Security & Privacy

3. Click Open Anyway button.

4. Make a choice and click ok.

Clean Caches

If the prompt in the picture appears, click OK, and then enter the power-on password, click OK.

When finish clean up, all caches file will remove to the Trash, you can choice clean up or not.

All Done!

Uninstall Residual Files

It will delete the remaining files uninstalled by Avast and Zoom on the Mac, but I will continue to update the script.

System Optimization

Turn on performance mode, improve Mac performance. Create a Launch Agents, every 30 minutes will auto release RAM and clean up DNS caches.

Check 404 Links

How to use it

1. Run the program, then put the link in the box and click continue. If you run it for the first time, the software will ask you to enter the password.

2. The notification will prompt, please wait for 5 to 10 minutes, if is a large website will cost more time, the program still run at background.

3. When the notification prompt is complete, the program will open "404 Links.txt" file. If nothing there it's mean there is no 404 links.

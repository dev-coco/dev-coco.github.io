常用术语

  1. 电子表格

    由多个工作表组成，用于存储数据和处理数据的一个文件。

  2. 工作表

    由多个单元格组成，显示在电子表格窗口中的表格。如果电子表格相当于一本书，那么工作表就相当于书中的一页纸。

  3. 电子表格ID

    在链接的 /d/ 到斜杆之间的字符。

    例子：https://docs.google.com/spreadsheets/d/XXXXXX/edit

  4. 工作表ID

    在链接最末尾 gid= 后面的数字。

    例子：https://docs.google.com/spreadsheets/d/XXXXXX/edit#gid=12345

  5. 分表引用

    在当前工作表中引用其它的工作表单元格。例如在工作表1中引用工作表2的A3单元格。

    公式：'工作表2'!A3

  6. 跨表引用

    使用 IMPORTRANGE 函数引用其它电子表格的工作表中的单元格内容。

    公式：IMPORTRANGE("XXXXXXX", "工作表1!A1")

错误类型

  1. #NULL!

    目前来看，这是一个无法重现的错误，但是确实是存在的。

  2. #DIV/0!

    任意数字除以 0 将会提示 #DIV/0! 错误，例如 =3 / 0，或者除以一个空白单元格就会出现这个错误。

  3. VALUE

    最常见的原因是公式中需要数值格式，但是内容中缺包含空格，文本或者其它的字符。

  4. #REF!

    引用了一个无效的单元格的时候会出现这种错误，或者引用的单元格已被删除。

  5. #NAME?

    无法识别公式中的函数名称。通常是在引用了一个单元格后，单元格被删除了。

  6. #NUM!

    公式中包含无效数值的时候会出现此错误。

  7. #N/A

    通常在公式中查找不到内容，或者在函数中缺少参数。

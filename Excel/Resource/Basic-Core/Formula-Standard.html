公式标准样式

一个标准的样式让公式看起来更美观，有助于理解多个函数嵌套的公式。在阅读公式和团队协作共同编辑公式的时候，显得特别重要。

函数名称全部使用大写

错误例子：

  • Sum(A1)
  • sum(A:A)
  • ifError(B13)

正确例子：

  • SUM(A1:A15)

逗号后面要留一个空格

错误例子：

  • SUM(A1,B1,C1,D1)

正确例子：

  • SUMPRODUCT(A2:A4, B2:B4)

比较运算符和算数运算符前后需要留一个空格

错误例子：

  • IFS(AND(B2>=80,B2<90),"中等水平",B2<80,"差生",B2>90,"优等生")
  • IFERROR(C3/D3,"error")

正确例子：

  • IFS(AND(B2 >= 80, B2 < 90), "中等水平", B2 < 80, "差生", B2 > 90, "优等生")
  • IFERROR(C3 / D3, "error")

分号后面要换行

错误例子：

  • {IMPORTRANGE("XXXXXXXXX", "工作表1!A1");IMPORTRANGE("XXXXXXXXX", "工作表1!A2");IMPORTRANGE("XXXXXXXXX", "工作表1!A3");IMPORTRANGE("XXXXXXXXX", "工作表1!A1")}

正确例子：

  • {IMPORTRANGE("XXXXXXXXX", "工作表1!A1");
    IMPORTRANGE("XXXXXXXXX", "工作表1!A2");
    IMPORTRANGE("XXXXXXXXX", "工作表1!A3");
    IMPORTRANGE("XXXXXXXXX", "工作表1!A1")}

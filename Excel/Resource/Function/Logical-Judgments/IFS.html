IFS

作用：判断多个条件，返回条件为 TRUE 的值。

语法：IFS(条件1, 返回值1, 条件2, 返回值2, ...)

当第一个条件返回 FALSE 时，就会判断第二个条件，以此类推，直到返回到正确的条件，如果所有条件都是 FALSE，会输出 #N/A。

例子：

运用：跟进成绩分类优等生、差生和中等水平。

需要计算的条件：

  1. 大于等于 80 分，小于 90 分，返回中等水平。
  2. 小于 80 分，返回差生。
  3. 大于 90 分，返回优等生。

在判断多个条件的时候，如果使用 IF 函数就需要一个函数套一个函数，IF(IF(IF...))，不美观，而且出错的时候不太好维护，可以使用 IFS 函数来代替。

使用 AND 函数判断两个条件，是否大于等于 80 分，又小于 90 分，再加上第二个条件，如果分数低于 80 分就返回差生，如果分数高于 90 分返回优等生。

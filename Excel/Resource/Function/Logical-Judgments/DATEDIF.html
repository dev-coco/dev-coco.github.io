DATEDIF

作用：计算两个日期之间的天数、月数和年数。

语法：DATEDIF(开始日期, 结束日期, 单位)

  • 单位
    • Y：返回起始日期和结束日期之间的整年数。
    • M：返回起始日期和结束日期之间的整月数。
    • D：返回起始日期和结束日期之间的天数。
    • MD：返回起始日期和结束日期之间的天数。忽略日期中的月份和年份。
    • YM：返回起始日期和结束日期之间的整月数。忽略日期中的天和年份。
    • YD：返回起始日期和结束日期之间的天数。忽略日期中的年份。

例子：

运用：计算两个日期之间有多少个星期。

使用 DATEDIF 计算两个日期的天数，设置两个日期，单位 用 D，然后再除以 7，这样就算出有多少个星期，但是有时候会出现除不尽或者小数的情况，使用 TEXT 函数，将格式设置为整数输出。

SWITCH

作用：设置一个内容，多个条件匹配，返回条件为 TRUE 的值，支持多行/多列数组输出返回值。

语法：SWITCH(范围, 条件1, 返回值1, 条件2, 返回值2, ..., 默认值)

  • 范围*：被匹配的范围。
  • 条件*：通过条件匹配范围，可以是数字，文本，运算符。
  • 默认值：没有相符结果的时候会返回默认值。

当第一个条件返回 FALSE 时，就会判断第二个条件，以此类推，直到返回到正确的条件。如果没有设置默认值并且条件都是 FALSE，会输出 #N/A。

例子：

运用：根据不同的职位给予不同的工资，经理 工资 3500 美元，店长 工资 3200 美元。

使用 SWITCH 函数，设置 B 列进行匹配，如果 B 列的内容是 经理 返回 3500，如果 B 列的内容是 店长 返回 3200，SWITCH 支持数组输出。

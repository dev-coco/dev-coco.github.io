IF

作用：判断一个条件的结果，自定义 TRUE 和 FALSE 返回值的输出结果。

语法：

  1. IF(条件, TURE 时返回值)
  2. IF(条件, TURE 时返回值, FALSE 时返回值)
  • 条件*：需要判断的逻辑条件。
  • TURE 时返回值*：当条件返回 TRUE 时输出的内容。
  • FALSE 时返回值：当条件返回 FALSE 时输出的内容。

例子：

运用：跟进成绩分类优等生、差生和中等水平。

需要计算的条件：

  1. 大于等于 80 分，小于 90 分，返回中等水平。
  2. 小于 80 分，返回差生。
  3. 大于 90 分，返回优等生。

使用 AND 函数判断两个条件，是否大于等于 80 分，又小于 90 分，如果是的话用 IF 函数返回中等水平。因为IF函数只能判断一个条件的 TRUE 和 FALSE，但是我们又有多个条件，那么在 FALSE 的条件内就需要再做一个 IF 函数的判断。那么现在已经做好了第一个条件的判断，在第二个 IF 内可以判断成绩小于 80 返回差生，剩下最后一个条件，可以不用再做判断了，在 FALSE 的结果可以直接返回优等生，因为不满足于第一个和第二个条件只有第三中情况，没有其他的条件了。

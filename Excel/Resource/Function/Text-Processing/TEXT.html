TEXT

作用：按照指定格式将数字转换成文本。

语法：TEXT(数值, 格式)

  • 数值：将要被转换的数值、日期或者时间。
  • 格式：
    • 0：代表数值少于格式指定的位数时，以 0 填充。当数值的小数超过指定的格式位数时，结果将会被四舍五入。
    • #：类似0, 但是会忽略头尾的0。
    • d：一位或两位数的月内几号。
    • dd：两位数的月内几号。
    • ddd：星期几的简称。
    • dddd：星期几的全称。
    • m：一位或两位数的月份值，或在时间中代表分钟值。除非此代码是与作为时间部分的小时或秒一起提供的，否则或在时间中代表分钟值月份。
    • mm：两位数的月份值，或在时间中或在时间中代表分钟值分钟值。除非此代码是与作为时间部分的小时或秒一起提供的，否则或在时间中代表分钟值月份。
    • mmm：月份的简称。
    • mmmm：月份的全称。
    • mmmmm：月份的第一个字母。
    • yy：两位数的年份值。
    • yyyy：四位数的年份值。
    • HH：24 小时制的小时值。
    • hh：12 小时制的小时值。
    • ss：时间的秒钟值。
    • ss.000：时间的毫秒值。
    • 上午/下午用于显示 12 小时制的小时值，并根据具体时间显示上午或下午。

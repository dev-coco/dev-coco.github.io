CHAR

作用：按照十进制 Unicode 编码 转换为对应字符。Unicode 编码参考资料：链接

语法：CHAR(数字)

例子：

运用1：使用字母生成字母序号。

A 的十进制 Unicode 编码是 65，那么使用 ROW 函数引用 A65，返回结果为 65，然后用 CHAR 函数计算 ROW 函数计算出来的值。因为是相对引用，当公式往下复制的时候，行数就会自动递增，这样就可以生成出字母序号了。

运用2：在文本中换行。

在文本需要换行的地方，使用 & 连接 CHAR 函数，因为换行的十进制的 Unicode 编码是 10，所以设置 CHAR 函数的值为 10，就可以实现在文本换行的效果了。

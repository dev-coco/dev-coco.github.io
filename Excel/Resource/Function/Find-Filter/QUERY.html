QUERY

作用：使用 SQL 语法对数据进行查询。QUERY 语法参考资料： Google Visualization API

语法：QUERY(范围, 查询, 标题数)

  • 范围*：需要查询的范围。
  • 查询*：SQL 查询语法。
  • 标题数：可选，数据上方标题的行数。

下面的数据表进行演示。

运用1：对范围中的 A 到 C 列的位置按照 A, C, B 列进行排序。

使用 SELECT 语句根据需求选择 A, C, B 列的数据。

运用2：选择 A 和 C 列的内容，C 列的内容按照倒序排序。

使用 SELECT 语句选择 A 和 C 列的数据，再使用 ORDER BY 语句选择 C 列，最后用 DESC 关键词进行倒序排序。

运用3：选择 A 到 E 列的内容设置成数组，查询 A 列包含张三的内容。

引用数据表 A 到 E 列并且设置成数组，使用 SELECT 语句选择所有的内容，再使用 WHERE 语句按照条件查询，因为引用的范围是数组，所以需要用 Col 选择列数，A 列在引用范围是第 1 列，设置 Col1 等于张三，因为张三是文本的内容，需要用单引号引用。

OFFSET

作用：指定单元格开始计算偏移的行或者列数，返回范围的引用内容。

语法：OFFSET(起始位置, 偏移列数, 偏移行数, 返回高度, 返回宽度)

  • 起始位置*：设置一个位置，从这个位置开始计算偏移。
  • 偏移列数*：从起始位置开始列算偏移的列。
  • 偏移行数*：从起始位置开始列算偏移的行。
  • 返回高度：从偏移的位置开始计算高度。
  • 返回宽度：从偏移的位置开始计算宽度。

例子：

运用：从当前位置获取往左一个单元格再往上一个单元格的内容。

首先，需要计算当前单元格的位置，使用 ROW 函数和 COLUMN 函数得出当前位置的行和列，然后用 ADDRESS 通过当前行和列的值输出文本范围。用 INDIRECT 函数引用 ADDRESS 输出的文本范围，这样就引用了当前的行和列。最后用 OFFSET 函数从当前的行和列作为起始的位置开始计算，行和列都 -1，这样就得出来当前往左并往上一个单元格的内容了。

INDEX

作用：根据行和列偏移返回指定范围的内容。

语法：INDEX(范围, 行数, 列数)

  • 范围*：被搜索的范围。
  • 行数*：返回范围中索引的行数。
  • 列数*：返回范围中索引的列数。

例子：

运用：查找王五对应的蔬菜是什么。

这里可以使用 INDEX 函数搭配 MATCH 函数进行更高级的数据查询。

语法：

  1. INDEX(范围, MATCH(条件, 范围, 搜索类型), 行数)
  2. INDEX(范围, MATCH(条件, 范围, 搜索类型), MATCH(条件, 范围, 搜索类型))

    通过 MATCH 函数计算出指定条件在范围中的行数或者列数，然后通过 INDEX 进行索引。

使用 MATCH 函数计算出王五在 B 列出现的行数。由于 B 列有名字，D 列有需要查找的内容，所以设置的范围需要同时包含着两列的信息，在 INDEX 函数设置范围 B2:B，然后把 MATCH 函数计算的结果作为行数。INDEX 函数还需要设置列数，前面设置的范围是 B, C, D，需要匹配的内容是在 D 列，也就是范围中的第三列，那么 INDEX 的列数需要设置成 3。

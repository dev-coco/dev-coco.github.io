VLOOKUP

作用：在一个范围内，从上到下按行正向查找某个值。

语法：VLOOKUP(搜索值, 范围, 索引, 排序)

  • 搜索值*：需要搜索的值，可以是数字，文本，运算符。
  • 范围*：需要匹配内容的范围。
  • 索引*：范围中包含返回值的列数。例如 B:D，B 是第一列，C 是第二列，D 是第三列。

  • 排序：默认为 TRUE。返回搜索结果（小于或等于搜索值）最接近的匹配项。

    如果设置 FALSE 将进行完全匹配（内容必须一致）。如果匹配到多个值，只会返回匹配到的第一个值对应的内容。

运用1：VLOOKUP 函数逆向查找数字 8 对应的内容。

VLOOKUP 只能正向查找，但是可以用一些方式来实现逆向查找的效果，但并不是真正意义上的逆向查找。

设定查找的内容数字 8 ，因为是数字格式，不是文本，所以这里不需要用到双引号。在 VLOOKUP 的范围需要重新一个范围，用 {1, 0} 新建一个数组， 1 代表搜索的列，0 代表搜索的结果列。然后 IF 函数判断结果也需要设置一个数组，把需要搜索的 B 列放在前面，查找结果的 A 列放在后面，这样 IF 函数判断 B 列如果搜索到内容就会返回 A 列的内容，就相当于把这两列的内容，重新生成一个表，顺序倒过来了，这样就可以让 VLOOKUP 正向查找了，因为顺序倒过来了，查找的结果就在第二列，索引的列就需要设置成 2。

运用：多列内容同时输出。

使用 VLOOKUP 匹配的时候，需要输出多个索引列，可以将多个列设置成数组，再搭配 ARRAYFORMULA 函数输出，这样就可以一次性输出多列的内容。

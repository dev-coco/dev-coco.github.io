IMAGE

作用：在单元格内容插入一张图片。

语法：IMAGE(链接, 模式, 高度, 宽度)

  • 模式：默认为1。
    • 1：不改变原图尺寸，自适应单元格的尺寸。
    • 2：会改变图片的比例，拉伸或者压缩图片，填满单元格。
    • 3：保持原图的尺寸在单元格中显示，可能会被裁剪。
    • 4：自定义图片大小。
  • 高度、宽度：只有模式的值为 4 的时候才生效。

例子：

运用：通过 Facebook 用户的 ID 获取头像。

Facebook 有一个公开的获取头像的 API 接口。

API：https://graph.facebook.com/用户ID/picture?width=1080&access_token=2712477385668128|b429aeb53369951d411e1cae8e810640

只需要将中间的用户 ID 的部分，引用 A2 的内容，再使用 IMAGE 函数返回图片就可以获取到头像了。

ADDRESS

作用：以文本格式返回单元格的引用范围

语法：ADDRESS(行, 列, 引用模式, 样式, 工作表)

  • 行*：引用的行数
  • 列*：引用的列数
  • 引用模式：
    • 1：默认值。绝对引用行和列。
    • 2：绝对引用行，相对引用列。
    • 3：相对引用行，绝对引用列。
    • 4：相对引用行和列。
  • 样式：
    • A1 样式：数字代表行数，字母代表列数，例如 B3 代表第二列的第三行。
    • R1C1 样式：R 的全称是 Row，代表行的意思，C 的全称是 Column 代表列的意思。例如 R3C2 同等于 B3，代表第二列的第三行。
  • 工作表：引用指定工作表的名称。

例子：

运用：获取当前位置。

使用 ROW 和 COLUMN 函数获取当前位置的行和列，再用 ADDRESS 输出，当前行和列的引用结果。
